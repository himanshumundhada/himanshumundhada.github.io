Who I Am

I am a Senior Data Engineer specialising in AWS-native data lake architecture, open table formats, and governance-first data platform design. My work sits at the intersection of distributed data processing, cloud infrastructure, and enterprise data governance — building systems that are not just scalable but auditable, secure, and analytically powerful.

My architectural approach centres on the Medallion Architecture — Raw (landing zone), Exploratory (curated transformations), and Curated (business-ready) layers — implemented on AWS using Apache Iceberg, Amazon S3 Tables, and AWS Glue Data Catalog with full schema evolution and ACID transaction support.

On the governance side, I implement AWS Lake Formation for fine-grained column/row-level access control, Amazon Macie for automated PII detection and sensitive data classification, and AWS Clean Rooms for enabling secure cross-team analytics without data exposure.

I also drive AI enablement through AWS Kiro (AI-assisted data engineering and infrastructure automation), integrated observability via AWS Distro for OpenTelemetry (ADOT), and participate in DevOps agent workflows for automated pipeline provisioning and deployments.
